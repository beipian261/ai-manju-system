# Landing Page 重新设计

## 概述

将首页（`src/app/page.tsx`）从"创建项目表单 + 简易 Hero"改造为纯粹的 Landing Page，传递产品的创意趣味感，引导用户点击 CTA 进入 Dashboard。

## 设计目标

- 去掉现有的创建项目表单
- 突出产品价值：AI 漫剧创作平台
- 用四步流程展示核心创作管线
- 用特色卡片展示关键功能亮点
- 整体风格：创意、有趣、干净

## 页面结构

### 1. 导航栏（极简）

```
[🟢 Logo: AI 漫剧]                    [🎬 开始创作]
```

- 左侧 Logo：渐变方块 + 文字，可点击回到首页
- 右侧 CTA 按钮：`btn-primary` 风格，跳转到 `/dashboard`
- 无其他导航项，保持 Landing Page 的聚焦

### 2. Hero 区

```
🎬  用 AI，把你的故事变成漫剧
    从创意到成片，一站式 AI 创作平台

    [🎬 免费开始创作 →]
```

- 大标题：`text-4xl md:text-5xl font-bold`，首行黑色，第二行 emerald 渐变
- 副标题：`text-base text-ink-secondary`，居中
- CTA 按钮：`btn-primary btn-lg`，跳转 `/dashboard`
- 背景装饰：在 Hero 区域加入一个**淡色的大号 emerald 渐变装饰圆形**（用 CSS absolute 定位），增加视觉层次感
- 加一个入场动画：标题/副标题/按钮依次 `animate-slide-up` + `stagger` 延迟

### 3. 创作流程（四步）

```
从灵感到成片，只需四步

 ┌──────┐   ┌──────┐   ┌──────┐   ┌──────┐
 │  1   │   │  2   │   │  3   │   │  4   │
 │🖊️ 写故事│   │🎭 创角色│   │🎬 画分镜│   │🎞️ 生成作品│
 └──────┘   └──────┘   └──────┘   └──────┘
```

- 四列网格 `grid-cols-1 md:grid-cols-4`
- 每个步骤卡片包含：**数字圈（emerald 渐变）** + **emoji 图标** + **标题** + **一句话说明**
- 卡片：`card` 风格，hover 时上浮，数字圈放大
- 步骤解释文字简明：
  - 🖊️ 写故事 — 输入创意灵感，AI 自动生成剧本
  - 🎭 创角色 — 设计角色形象，AI 保持跨场景一致性
  - 🎬 画分镜 — 智能拆解场景，自动生成画面分镜
  - 🎞️ 生成作品 — 合成视频配音，一键发布多平台

### 4. 功能特色（2×2 网格）

```
✨ 核心功能

 ┌─────────────────┐  ┌─────────────────┐
 │ 🤖 AI 剧本工坊   │  │ 🎯 角色一致性   │
 │ 输入创意，自动   │  │ 跨场景保持角色   │
 │ 生成完整剧本     │  │ 外貌服装统一     │
 └─────────────────┘  └─────────────────┘
 ┌─────────────────┐  ┌─────────────────┐
 │ 🎬 AI 导演模式   │  │ 📤 多平台发布   │
 │ 智能分析场景，   │  │ 一键导出适配     │
 │ 给出构图/镜头建议 │  │ 抖音/B站/小红书  │
 └─────────────────┘  └─────────────────┘
```

- 2×2 网格 `grid-cols-1 md:grid-cols-2`
- 每个特色卡片：左上角 emerald 装饰条 + emoji + 标题 + 描述
- hover 时整体上浮 4px，装饰条变色

### 5. 底部 CTA

```
💪  准备好了吗？
    开始创作你的第一部 AI 漫剧
    [✨ 立即开始创作 →]
```

- 居中排版，emerald 渐变装饰背景块
- 突出显示的 CTA 按钮

### 6. Footer

```
© 2026 AI 漫剧系统
```

- 极简版权信息
- 居中，小字，灰色

## 视觉设计

- **主色**：`#10B981` (emerald-500)，配套渐变 `from-emerald-500 to-emerald-600`
- **背景**：纯白 `#FFFFFF`，Hero 区域有淡色装饰
- **字体**：`DM Sans` 为主，标题粗体
- **卡片**：12px 圆角 (`rounded-card`)，1px 边框 `#F0F0F0`，阴影 `0 1px 3px rgba(0,0,0,0.04)`
- **动画**：`animate-slide-up` 配合 `stagger-1` 到 `stagger-4`
- **装饰元素**：Hero 区加入一个大型半透明 emerald 渐变圆（position absolute），增加视觉深度

## 不包含的内容

- ❌ 创建项目表单
- ❌ 作品展示画廊（无实际数据）
- ❌ 用户评价/Testimonials
- ❌ 定价/套餐信息

## 技术实现

- 只修改 `src/app/page.tsx` 一个文件
- 复用现有的 CSS 组件类（`card`, `btn-primary`, `btn-lg`, `text-gradient`, `animate-slide-up`, `stagger-*`）
- 不需要新增 CSS 或组件
- 保持 `'use client'` 指令（目前页面使用 useState/useEffect 管理创建表单状态，改造后不再需要，可改为服务端组件，但为保持简单暂时保留 `'use client'`）

## 前后对比

| 项目 | 改造前 | 改造后 |
|------|--------|--------|
| Hero 区 | 简单标题 + 按钮 | 渐变标题 + 装饰 + 动画 |
| 创建表单 | 直接放在首页 | ❌ 移除 |
| 内容结构 | 顶部 Hero + 表单 | 导航 + Hero + 流程 + 特色 + CTA + Footer |
| 产品介绍 | 无 | 四步流程 + 四大特色 |
| 导航 | Logo + 开始创作 | Logo + 开始创作（不变） |
