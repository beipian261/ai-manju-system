# AI 漫剧系统 — UI 设计规范

> **日期**: 2026-06-21
> **状态**: 已批准
> **设计方向**: 纯白极简 + 翡翠绿强调

---

## 1. 设计理念

**纯白极简 · 翡翠点睛** — 以 Apple 式的纯净白色为基底，用翡翠绿（Emerald）作为唯一的品牌强调色。追求极致的干净、通透与专注，让内容成为视觉的主角。

### 设计原则

| 原则 | 说明 |
|------|------|
| **内容优先** | 大量留白、克制装饰，让用户聚焦于创作内容 |
| **翡翠点睛** | 绿色仅在关键交互元素（按钮、活跃态、进度指示）上使用 |
| **细腻质感** | 薄边框、微阴影、柔和圆角，营造精致触感 |
| **一致节奏** | 统一的间距、圆角、字体梯度，形成规律感 |

## 6. 页面设计

### 6.1 首页 (Landing Page)

**布局结构**:
1. **导航栏** — Logo（翡翠绿渐变方块 + 文字）+ 登录 + 开始创作按钮
2. **Hero 区域** — 标签徽章（AI 驱动·全流程自动化）→ H1 大标题（渐变绿"漫剧故事"）→ 副标题 → 双 CTA 按钮
3. **功能卡片** — 四列网格：智能剧本 / 自动分镜 / AI 绘画 / 视频合成
4. **创作流程** — 四步骤，第一步数字为翡翠绿实心，其余为浅绿空心
5. **CTA 横幅** — 浅绿渐变背景 + 绿边框圆角卡片
6. **页脚** — 版权信息

### 6.2 仪表盘 (Dashboard)

**布局结构**:
1. **左侧导航** — Logo + 导航项（仪表盘/项目/角色/设置）
2. **顶栏** — 欢迎文字 + 新建项目按钮
3. **统计卡片** — 四列网格（项目总数/角色数量/剧本完成/已发布），最后一张翡翠高亮
4. **快速操作** — 三列卡片
5. **最近项目** — 三列项目卡片（图标 + 标题 + 类型 + 状态标签 + 时间）

### 6.3 项目详情页

**布局结构**:
1. **顶栏** — 返回按钮 + 项目名 + 状态标签 + 操作按钮
2. **Tab 导航** — 概览 / 剧本 / 分镜 / 角色 / 配音 / 时间线 / 审核 / 发布
3. **概览内容**:
   - 项目信息三列（题材/风格/时间）
   - 进度条（翡翠绿渐变）
   - 管线四步骤卡片（已完成绿色实线/进行中绿虚线/待开始灰色）
   - 故事简介区域

### 6.4 登录页

**布局结构**:
1. **左栏（品牌区）** — 浅绿渐变背景 + Logo + 系统名 + 标语
2. **右栏（表单）** — 登录标题 + 邮箱/密码输入 + 记住我/忘记密码 + 登录按钮 + SSO 登录 + 注册链接

## 8. 实施指引

### Tailwind 配置变更

需要在 `tailwind.config.js` 中更新或添加以下配置：

```js
colors: {
  // 保留现有 emerald 色系
  // 添加 surface/border 语义色别名
  surface: {
    DEFAULT: '#FAFAFA',
    dark: '#27272A',
  },
  border: {
    DEFAULT: '#F0F0F0',
    strong: '#E4E4E7',
    dark: '#3F3F46',
  },
}
```

### 需替换的 CSS 文件

1. `src/app/globals.css` — 保留动效和工具类，替换当前 purple/pink 色系为 emerald 绿
2. `src/app/page.tsx` — 首页整体替换为新设计
3. `src/app/dashboard/page.tsx` — 仪表盘整体替换
4. `src/app/login/page.tsx` — 登录页替换
5. `src/components/navbar.tsx` — 导航栏统一风格
6. `src/components/ui/*` — 各 UI 组件逐步更新
